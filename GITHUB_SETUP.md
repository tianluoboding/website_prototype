# 🐙 GitHub设置指南

## 📋 步骤1: 在GitHub上创建新仓库

### 1️⃣ 登录GitHub
访问 https://github.com/tianluoboding

### 2️⃣ 创建新仓库
1. 点击右上角的 `+` 号
2. 选择 `New repository`
3. 填写仓库信息：
   - **Repository name**: `website_prototype`
   - **Description**: `WattUp Website - 智能骑行解决方案营销网站`
   - **Visibility**: `Public` (这样GitHub Pages才能免费使用)
   - **不要**勾选 "Add a README file"
   - **不要**勾选 "Add .gitignore"
   - **不要**勾选 "Choose a license"
4. 点击 `Create repository`

## 📋 步骤2: 连接本地仓库到GitHub

### 在终端运行以下命令：

```bash
# 添加远程仓库
git remote add origin https://github.com/tianluoboding/website_prototype.git

# 推送代码到GitHub
git push -u origin main
```

## 📋 步骤3: 启用GitHub Pages

### 1️⃣ 进入仓库设置
1. 在GitHub仓库页面，点击 `Settings` 标签
2. 左侧菜单找到 `Pages`

### 2️⃣ 配置Pages
1. **Source** 选择 `GitHub Actions`
2. 点击 `Save`

### 3️⃣ 等待自动部署
1. 点击 `Actions` 标签
2. 查看部署进度
3. 通常需要2-5分钟完成

## 📋 步骤4: 访问网站

### 🌐 网站地址
**https://tianluoboding.github.io/website_prototype/**
